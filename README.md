# LightFM Movie Recommendation System

A hybrid recommendation pipeline built on the MovieLens 100K dataset, implementing collaborative filtering using the LightFM library under the guidance of Prof. Bidyut Kumar Patra, IIT (BHU) Varanasi.

## Features
- User-based collaborative filtering
- Item-based collaborative filtering  
- Matrix factorization using LightFM (WARP loss)

## Tech Stack
Python, LightFM, pandas, NumPy

## Dataset
MovieLens 100K — 100,000 ratings from 943 users on 1,682 movies.

## Setup
```bash
pip install -r requirements.txt
python main.py
```
